## 👨‍💻 Sobre mim

Sou **Desenvolvedor Full Stack** com foco no ecossistema **JavaScript/TypeScript**, criando aplicações web completas do front ao back.

Tenho experiência prática em:
- 🔐 Autenticação de usuários (JWT, NextAuth)
- 🔌 Consumo e criação de **APIs REST**
- ⚙️ Gerenciamento de estado com **React Query**
- 🗄️ Integração com banco de dados (**Prisma, Supabase, MongoDB**)
- 🚀 Deploy de aplicações em produção (**Vercel**)
- 🧪 Testes E2E com **Cypress** e pipelines com **GitHub Actions**

> 🎯 Atualmente buscando minha **primeira oportunidade como Desenvolvedor Júnior**, onde possa contribuir com projetos reais e crescer com o time.
