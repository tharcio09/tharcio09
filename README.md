## 👨‍💻 Sobre mim

**Desenvolvedor Fullstack** — construo aplicações web completas com foco em fluxo real de usuário, regras de negócio e deploy em produção.
Projetos reais em produção (principais):

- 🔐 **HelpFlow** — sistema de help desk com autenticação por sessão, tickets, banco relacional e CI/CD
- 🔗 **DevLinks** — plataforma de links com API REST, upload em nuvem e testes E2E
- 📊 **Crypto Dashboard** — dashboard SSR com dados em tempo real
- 📱 **Lista de Mercado** — PWA mobile-first com suporte offline completo

Minha base vem de **diagnóstico técnico e controle de processos** — hoje aplico essa abordagem diretamente na resolução de problemas e debugging.

> 🎯 Disponível para estágio ou posição júnior em desenvolvimento fullstack.
